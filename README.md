# Python学生成绩管理系统 python753

## 项目概述

Python学生成绩管理系统是一款基于Python语言和Django框架开发的应用，旨在帮助学校管理学生成绩和相关信息。系统提供了一系列功能，包括但不限于学生信息管理、成绩管理、教师与班级管理、以及个人信息修改等。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Django
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python版本：3.0及以上
- MySQL版本：8.0

## 功能模块

- **学生信息管理**：对学生信息进行增删改查操作。
- **学生成绩管理**：记录和修改学生成绩，支持考试得分查询。
- **教师管理**：管理教师信息，包括但不限于信息的增删改查。
- **班级管理**：对班级信息进行管理。
- **个人信息管理**：用户可以修改个人信息和密码。

## 系统角色

- 系统管理员
- 教师
- 学生
